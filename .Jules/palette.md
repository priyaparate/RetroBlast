## 2026-01-14 - Auth Form Feedback Pattern
**Learning:** Retro-themed interfaces often sacrifice standard usability conventions (like clear labels and error messages) for style.
**Action:** When working with highly stylized "retro" forms, implement error feedback using existing theme elements (like the flickering `.title` class) to maintain immersion while restoring critical feedback loops. Use inline styles sparingly to adjust sizing without cluttering the global stylesheet.
